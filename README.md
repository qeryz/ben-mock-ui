- Next.js 15 (App Router)

# Project Structure

```
src/
├── app/              # Next.js App Router
├── components/       # Well-organized component hierarchy
├── stores/           # State management via Zustand
├── hooks/            # Custom hooks
├── data/             # Mock data (to be replaced with APIs)
└── lib/              # Utilities
```
